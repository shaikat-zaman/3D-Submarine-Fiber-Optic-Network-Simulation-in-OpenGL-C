# OptiFlux: Cross-Border Network Simulation 🌊🌐

![OpenGL](https://img.shields.io/badge/OpenGL-2D89EF?style=for-the-badge\&logo=opengl\&logoColor=white)
![C](https://img.shields.io/badge/C-00599C?style=for-the-badge\&logo=c\&logoColor=white)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

OptiFlux is a real-time 3D simulation built in **C & OpenGL (GLUT)**.
It visualizes submarine fiber optic cable communication between Bangladesh and Singapore.

## ✨ Features

* Bezier Curve based cable simulation
* Ocean wave animation
* Real-time packet flow
* Cable break & repair system
* Day/Night mode

---

## 🛠️ Tech Stack

* C
* OpenGL (GLUT)

---

## 🚀 Run

```bash
gcc src/*.c -o app -lGL -lGLU -lglut
./app
```

---

## 🎮 Controls

* S → Send Packet
* B → Break Cable
* R → Repair Cable
* D → Day/Night
